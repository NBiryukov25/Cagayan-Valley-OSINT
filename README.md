Cagayan Valley OSINT Tool

Location-specific open source intelligence tool for Tuguegarao City, Cagayan Province, and Region II Philippines. Generates Google dorks targeting local news, government sites, universities, social media, and incident reports.

Features

Person and business searches across local platforms
Barangay-level community intelligence
Natural disaster tracking (typhoons, floods, earthquakes, landslides)
Accident reports (vehicular, maritime, fire, infrastructure)
Political event monitoring (elections, protests, corruption, violence)
Local term support for better results (bagyong, linog, aksidente, halalan, etc.)
Export to CSV with clickable Google URLs

Incident Categories

Disaster: PAGASA, NDRRMC, flood, typhoon, earthquake, landslide
Accident: PNP, BFP, vehicular, maritime, fire, construction
Political: Comelec, Ombudsman, election violence, protests, graft

Target Locations

Tuguegarao City
Cagayan Province
Cagayan Valley / Region II
Aparri, Lal-lo, Sanchez Mira, and other municipalities

Sources

Local News: Tribune, Manila Times, Newsbytes, ABS-CBN/GMA regional
Government: tuguegaraocity.gov.ph, cagayan.gov.ph, DILG, BIR, SEC
Education: CSU, SMCT, UCV, UST Cagayan
Social: Facebook, Twitter/X, Instagram, Reddit, Marketplace

Usage

python3 cagayan_osint.py

Follow prompts to enter target info. Select incident types to include. Output is a timestamped CSV with priority-sorted, clickable search URLs.

Requirements

Python 3.6+

No external dependencies.
